# Student Performance Analysis

## Overview

This project analyzes student performance data across multiple subjects (CAO, Data Science, and Java).
The goal is to identify patterns, compare results, and understand factors affecting student performance.

## Dataset

Real student data (self-collected)
The dataset is stored in an Excel file:

* Each subject is in a separate sheet (CAO, Data Science, Java)

## Tools Used

* Python
* Pandas
* Matplotlib

## Project Structure

* CAO → Subject analysis
* Data Science → Subject analysis
* Java → Subject analysis

## Steps

1. Data Cleaning (handling missing values and inconsistencies)
2. Data Analysis (calculating averages and comparisons)
3. Categorization (high, medium, low performance)
4. Data Visualization (charts and graphs)

## Results & Insights

* Identified performance differences between students
* Categorized students based on performance levels
* Discovered patterns affecting student results

## Conclusion
This project demonstrates data analysis skills on real-world data across multiple subjects.
